== Description ==

This plugin posts videos, title and description via the Hydravid Syndicate app.  It also enables your Wordpress site to become a part of the Hydravid Syndicate network, in turn, allowing you to post to other Wordpress sites.


== Installation ==

1. Upload `hydravid` directory to the `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your Hydravid username (email) and License ID where prompted

== Frequently Asked Questions ==

= How do I use the plugin? =

Once the plugin is installed, you can post to your blog via http://app.hydravid.com.
